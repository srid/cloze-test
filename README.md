# Cloze tests in Elm (WORK IN PROGRESS)

A single-page web app for [cloze tests](https://en.wikipedia.org/wiki/Cloze_test), written in Elm.

## Local development

```
elm-reactor
open http://localhost:8000/src/ClozeTest/Main.elm
```
